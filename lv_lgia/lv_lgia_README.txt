# lv_lgia_README.txt

The files in this section result from the conversion of datasets originating
from [LGIA](https://www.lgia.gov.lv/)

## Included grids

### Latvian vertical grid:

*Source*: [LGIA](https://www.lgia.gov.lv/en/latvijas-kvazigeoida-modelis)
*Format*: GeoTIFF converted from 'XYZ'
*License*: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)
*Credit*: (c) Latvijas Ģeotelpiskās informācijas aģentūra - Latvia
*Horizontal CRS*: EPSG:4949 (LKS-92)

Vertical transformation for Geoid model LV'14. Used to make the transition
from heights in vertical CRS Latvia 2000 height (EPSG:7700)
to heights above the ellipsoid in LKS-92 (EPSG:4949).

* lv_lgia_LV14.tif
